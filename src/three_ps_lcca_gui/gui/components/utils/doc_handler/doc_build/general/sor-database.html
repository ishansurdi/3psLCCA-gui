Material Suggestions (SOR Database)

A Schedule of Rates (SOR) database provides pre-filled material cost and carbon emission data throughout the project.

How It Works

When an SOR database is selected, the Material Dialog activates autocomplete suggestions. Start typing a material name to see matching items with the following fields pre-filled:

Field Source
Rate SOR unit cost
Emission Factor SOR carbon data
Unit SOR standard unit

Options

  • Select a database: enables suggestions based on your project country.
  • No suggestions: all material data must be entered manually.

[!NOTE] The SOR database only provides default values. All pre-filled fields remain editable.
