# Discount Rate

As per ISO/DIS 15686-5.2(en) *Buildings and constructed assets — Service-life planning — Part 5: Life cycle costing*:

Discount rate is the time value of money that is used to convert cash flows occurring at different times to a common time.

Alternatively, the discount rate can be defined as the interest rate used to convert future costs, benefits, or income streams into a single present-day value. It reflects the time value of money.
