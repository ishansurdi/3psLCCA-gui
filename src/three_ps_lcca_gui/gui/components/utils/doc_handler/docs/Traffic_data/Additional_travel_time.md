# Additional Travel Time

Additional travel time refers to the incremental temporal burden imposed on travelers due to construction, maintenance activities, or demolition.

$$Additional\ travel\ time = \frac{Additional\ reroute\ distance}{speed}$$

### Example
- **Additional reroute distance**: 0.175 km
- **Speed**: 20 km/hr
- **Calculation**: $\frac{0.175}{20} \times 60 = 0.525$ minutes
